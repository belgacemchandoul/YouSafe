# YouSafe 🦽

> **Coming Soon** · YouSafe is currently in its final stages before launch.

YouSafe is a community-focused web platform that maps wheelchair-accessible locations across Ireland, helping people with mobility needs discover and navigate accessible spaces with confidence.

---

## Features

- 🗺️ **Interactive Accessibility Map** · Browse wheelchair-friendly locations across Ireland on a fully interactive map
- 📍 **Location Profiles** · Detailed pages for each location including accessibility info, photos, and embedded media
- 🔍 **Search & Filter** · Find locations by category, region, or accessibility type
- 🖼️ **Media Support** · Each location supports image and video content for a richer experience
- ✍️ **Blog** · Articles and guides around accessibility, mobility, and inclusive spaces in Ireland
- 🔒 **Admin Dashboard** · Full content management system for moderating and managing locations, media, and blog posts
- 📱 **Mobile-First Design** · Fully responsive across all devices
- 🔎 **SEO Optimised** · Built with discoverability in mind
